Help

Click Edit Players to... edit the players. You can set each player to either Human or NPC. You can even set both players to NPC and watch the fun. If a player is set to Human you can change their name. Each player's Mark can also be changed (changing one automatically changes the other). Once you're happy click the confirm button to save the changes. Clicking confirm will reset the current win counts.

To start a game, click the Play button next to the Help button. When a game has been completed, you can click the Play button again to start a new round. When a new round starts the first player is chosen at random.

Quick note about the NPCs: they're dumb. They just randomly pick from an empty space.
